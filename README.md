# Roket

2D spaceship game with Newtonian physics.
© Vincent Laurila 2019-2025

## How to Develop

### Option 1: Node

- Install NodeJS 22 (other versions might very well work)
- Enable Corepack if needed: `corepack enable`
- `pnpm install`
- `pnpm run dev`

### Option 2: Docker

- Install Docker
- `./docker-build.sh`
- `./docker-run.sh`

## Testing

Run the unit tests with:

```bash
pnpm test
```
